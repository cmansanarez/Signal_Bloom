# Signal Bloom - A Modular AI-Assisted Live Visual Performance Environment

## Project Overview

This project explores the evolving relationship between performer, machine, and emergence through a modular AI-assisted live visual performance system.

Combining Hydra live coding, generative art systems, AI image/video generation, audio-reactive visuals, and immersive projection environments, the project investigates how live audiovisual performance can function as a site of improvisation, collaboration, instability, and transformation between human and machine systems.

Rather than relying solely on pre-authored media assets, the system aims to generate visual material dynamically during live performance. Through text prompts, recursive visual feedback, real-time coding, and machine-assisted image generation, the performance environment continuously evolves in response to performer interaction, sound, audience energy, and emergent system behavior.

The project exists at the intersection of:

* Live coding
* Generative art
* AI-assisted creativity
* Queer futurism
* Cybernetic aesthetics
* Glitch art
* Performance art
* Immersive media systems

## Core Problem / Purpose

### Problem Statement

Most contemporary AI-generated visual media is static, pre-rendered, and authored outside the moment of performance. This creates a separation between generation, improvisation, and audience experience.

This project asks:

What happens when AI generation itself becomes performative?

The project seeks to explore how generative systems can become active collaborators during live audiovisual performance rather than passive tools used before performance begins.
